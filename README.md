# Jellybean-Adventure-3D
Collect jelly beans, gain abilities and pass levels

# Game Description 
We're trying to complete a running platform with a total of four levels. There are four jellybeans and their abilities. You can't get more than one at a time. There's a limit, and they last for 3 seconds.

Red --> Increases our speed by 2x 

Blue --> Jumps 2x further 

Yellow --> Makes us invulnerable 

Special (Purple) --> A feature that includes all

# Controller 

Joystick for left and right control 

Button jump
